# Agent Pipeline

Claude Code slash commands for AI-driven feature development.

## Commands
- `/pipeline` — Full orchestrated pipeline: PRD → TechSpec → Tasks → Implement → QA
- `/criar-prd` — Generate a PRD with clarification questions
- `/criar-techspec` — Generate a TechSpec from a PRD
- `/criar-tasks` — Decompose PRD+TechSpec into ordered tasks
- `/executar-task` — Implement a single task with TDD + code review
- `/executar-review` — Code review against PRD/TechSpec/conventions
- `/executar-qa` — E2E QA via Playwright MCP
- `/executar-bugfix` — Reproduce, fix, test, review a bug
